# Stdlib Overview

This section provides an overview of the modules within this directory.

* [`base`](/api-reference/stdlib/base) - Basic stdlib data structures.

* [`chat`](/api-reference/stdlib/chat) - Chat primitives.

* [`genslot`](/api-reference/stdlib/genslot) - A method to generate outputs based on python functions and a Generative Slot function.

* [`instruction`](/api-reference/stdlib/instruction) - Instructions.

* [`mify`](/api-reference/stdlib/mify) - Mify classes and objects.

* [`mobject`](/api-reference/stdlib/mobject) - MObject.

* [`requirement`](/api-reference/stdlib/requirement) - Requirements are a special type of Component used as input to the "validate" step in Instruct/Validate/Repair design patterns.

* [`sampling`](/api-reference/stdlib/sampling) - sampling methods go here.

* [`session`](/api-reference/stdlib/session) - Mellea Sessions.
