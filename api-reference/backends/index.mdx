# Backends Overview

This section provides an overview of the modules within this directory.

* [`cache`](/api-reference/backends/cache) - Caching strategies.

* [`dummy`](/api-reference/backends/dummy) - This module holds shim backends used for smoke tests.

* [`huggingface`](/api-reference/backends/huggingface) - A backend that uses the Huggingface Transformers library.

* [`litellm`](/api-reference/backends/litellm) - A generic LiteLLM compatible backend that wraps around the openai python sdk.

* [`model_ids`](/api-reference/backends/model_ids) - Utilities for model identifiers.

* [`types`](/api-reference/backends/types) - Useful type definitions for models, formatters, and backends.

* [`watsonx`](/api-reference/backends/watsonx) - A generic WatsonX.ai compatible backend that wraps around the watson_machine_learning library.
